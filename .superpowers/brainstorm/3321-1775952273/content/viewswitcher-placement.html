ViewSwitcher: Where should the tab bar live?

The ViewSwitcher is the main UX innovation — it changes everything about the header

A

Integrated in DatabaseShell header

ViewSwitcher tabs sit directly inside the database note's header area, replacing the per-view title bar. Toolbar below tabs.

B

Floating above view content

ViewSwitcher floats as a sticky bar above the view. Tabs + toolbar merge into one horizontal strip. View content scrolls beneath.

C

Separate top-level row

ViewSwitcher gets its own dedicated row above everything — like Notion. Below it: toolbar, then view content. Most breathing room.
