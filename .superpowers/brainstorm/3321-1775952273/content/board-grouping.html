Board view — how should grouping work?

The group-by field determines which column each card falls into

A

Toolbar dropdown — "Group by: Field"

The toolbar has a "Group by" label + dropdown showing all eligible fields (Select, Multi-Select). Selecting a field instantly re-groups the board columns. No separate config panel.

B

Board header — "Group by" pill button

The active group-by field shows as a pill/chip in the board view's own header area (e.g. "Grouped by: Status ✕"). Clicking the chip opens a field picker popover inline.

C

Semi-automatic — detect from first Select field

Board automatically groups by the first Select/Multi-Select field it finds. No explicit grouping control. User can change it via the toolbar only.
