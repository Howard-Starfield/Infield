What "mark done" does to events
"Default done behavior" options
1. Everything active (Google style)
Events stay full color even when done. No visual change. User must check a separate "Done" filter to hide them.
2. Dim + strikethrough (default)
Events dim to 40% opacity with strikethrough title. Hover reveals undo. Click to open editor with Restore.
3. Move to "Done" view (Linear style)
Completed events disappear from the calendar grid and appear in a collapsible "Done" section below or in a filter panel. Keeps calendar clean.
