Key principles:
1. Hover → undo chip appears inline. No need to open editor for quick fixes.
2. Click → editor shows full context. "Restore" button brings it back to active state.
3. Dim, don't hide — completed events stay in their calendar position for context and review.
4. Board ↔ Calendar sync — if a card is marked done in Board, it dims in Calendar too.

Yes — hover preview + mark done

Hover shows preview tooltip (month & week). Events dim with strikethrough when marked done. Hover reveals undo chip. Click opens editor with Restore option. Board ↔ Calendar status stays in sync.

B

No hover preview — just mark done

Skip the hover tooltip. Click event → editor with done/restore. Simpler to build but less fluid.
