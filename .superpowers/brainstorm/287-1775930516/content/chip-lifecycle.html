⚠️ Where do deleted events go after the chip disappears?
① Delete — chip disappears after 4s
User taps Delete. Chip shows "deleted" toast → fades out. The event is gone from the calendar view. Where is it?
Answer: The row is NOT deleted from the database. It gets a status = "deleted" flag. It is simply hidden from all calendar views.
② Finding deleted events
Three places a user can find and restore a deleted event:
A. Unscheduled panel — events without a date. Deleted calendar events (which have a date) don't go here unless date was cleared.
B. Database Grid view — the grid shows ALL rows including deleted ones. There's a "Show deleted" filter toggle.
C. Trash / History section — a dedicated "Deleted events" section in the database sidebar (below the view tabs). This is the primary recovery path.
